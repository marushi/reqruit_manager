# reqruit_manager

就活管理を行うアプリ

## 機能

大きく**アドバイザー**と**就活生**に機能が分けられる(あくまで就活生メインのアプリ)

### 就活生

**・選考状況の確認**

→会社情報・メモの入力

→選考状況(選考・期限)の入力

→ESを登録


**・FB(FeedBack)機能**

※アドバイザーを指定することで利用可能


→FB依頼の作成(ESの添削・電話対応)

→FB履歴の確認(添削されたESなどが確認可能)


**・ニュース機能**

※アドバイザーを指定することで利用可能


→アドバイザーが発信した就活情報が閲覧できる


**・設定機能**


→自分のプロフィール情報が閲覧可能

→オープンES作成画面・プロフィール変更・アドバイザー検索への導線
